# Nmap Advanced Scan Lab

## Objective

Learn advanced Nmap scanning techniques for host discovery, service detection, and aggressive scanning.

## Commands

```bash
nmap -Pn localhost
nmap -A localhost
nmap -sS localhost
nmap -F localhost
```

## Expected Learning

- Skip host discovery with `-Pn`
- Perform aggressive scans with `-A`
- Understand SYN scans using `-sS`
- Use fast scanning with `-F`

## Notes

Some scan types may require root privileges.
